# Websocket target

Serveur réalisé en NodeJS
Client réalisé en TypeScript vers Js et HTML

## Démarrage du serveur

node server.js

## Client

* git clone 
* npm install
* tsc ; webpack-cli
* charger index.html dans un navigateur
